# 📊 Pilote de Trésorerie Freelance - Édition Expert

Une application web autonome et sécurisée pour gérer votre trésorerie micro-entrepreneur. Calculez votre bénéfice net réel en temps réel, gérez votre TVA et sécurisez vos données localement.

## 🚀 Points Forts
- **Calcul Fiscal Automatisé** : Gestion du seuil de TVA (36 800 €) et déduction automatique des cotisations sociales (21,1%).
- **Confidentialité Totale** : Utilisation de la technologie **IndexedDB**. Vos données financières et votre logo restent sur votre ordinateur et ne sont jamais envoyés sur un serveur.
- **Système de Backup** : Exportez et importez vos données via des fichiers JSON pour changer d'ordinateur sans rien perdre.
- **Rapports Professionnels** : Génération de PDF personnalisés avec votre logo pour vos archives ou votre comptable.
- **Interface Moderne** : Dashboard interactif avec mode sombre et design adapté aux écrans d'ordinateur.

## 🛠️ Installation Rapide
1. Créez un dépôt sur GitHub.
2. Téléchargez les fichiers `index.html`, `style.css` et `script.js`.
3. Activez **GitHub Pages** dans les réglages de votre dépôt pour obtenir votre lien URL privé.

## 🔒 Sécurité des Données
Contrairement aux outils classiques, ce dashboard n'utilise pas de base de données distante. Il crée une base de données structurée directement dans le "bac à sable" sécurisé de votre navigateur. 
*Avantages : Rapidité, accès hors-ligne et protection absolue de votre vie privée.*

## 📈 Technologies utilisées
- HTML5 / CSS3 (Variables & Flexbox)
- JavaScript Vanilla (ES6+)
- IndexedDB (Stockage local persistant)
- Chart.js (Visualisation de données)
- Html2Pdf.js (Génération de documents)
